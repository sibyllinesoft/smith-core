# Code of Conduct

## Our Pledge

We are committed to making participation in the Smith Core community a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and discuss ideas, not people.
- Accept feedback gracefully.

## Unacceptable Behavior

- Harassment, intimidation, or discrimination.
- Personal attacks, trolling, or derogatory comments.
- Publishing private information without permission.

## Enforcement

Project maintainers are responsible for clarifying and enforcing these standards and may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

## Scope

This Code of Conduct applies to all project spaces, including GitHub issues, pull requests, discussions, and other official communication channels.

## Reporting

Report violations by opening a private security advisory on GitHub or contacting the maintainers through repository admins.
